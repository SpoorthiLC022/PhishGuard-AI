# PhishGuard AI

AI-powered phishing website detection system built using Machine Learning.

## Features
- Detect phishing URLs
- Real-time URL analysis
- Streamlit web app
- Random Forest model

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## Accuracy
96.7%
